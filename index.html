Musicat

Simple music player for android

View on GitHub Download .zip Download .tar.gz

Goals

  • Musicat is an open source software and it will remain free for the rest of my life. I hope that someone will help me to improve the app

  • Musicat try to add some feautures that I did not see in other similar apps

Floating Bubble

Inspired by faceCook app (I hate facebook app :) , I implemented a floating bubble to manage basic action of music player. You can show or hide the bubble in settings. By default it is active. How to use:

  • Long press: you can drag on screen
  • Release finger: now gestures are enable
  • Swipe right: skip to next song
  • Swipe up: play/pause current track
    ***

Real Image Editing (Art Work)

Like other id3x tag, you can edit title, album, artist and image. App offers you the possibility to search an image form a web service. From my stats a title like "This is title_-_(HD) original_video feat xxx" corresponds at 90% to the searched image. If not, I'm so sorry. To improve speed of scroling I did chose to show art work from MediaStore. This means that correct image will show itself after a reboot or an automatic media resfresh caused from google play music or similar app. But editing of an image of a track causes the change immediatly into the file. Copy and paste the file on your pc to see.

Web server for player

You can listen music from your pc or any other devices that support wifi (this devices must be connected to the same network). To run server, go to the settings and enable "Web server". Musicat will show to you IP used". Open your favorite browser and put the local url. Musicat use port 8080. Tested on network with a normal firewall. Is not necessary to open the port
